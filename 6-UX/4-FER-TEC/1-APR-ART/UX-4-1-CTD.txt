Guias de estilo

Um guia de estilo contempla todas as diretrizes de design de uma instituição ou empresa. Ele pode ser aplicado, em menor escala, em produtos, serviços, marcas etc.

 

Um dos guias de estilo mais conhecidos é o Material Design, criado pelo Google em 2014.

UI kit

Um kit de interface do usuário abrange vários elementos de design, como componentes e estilos.

 

Os componentes trazem significado à interação do usuário com a interface, como botões e menus de navegação.

 

Já os estilos são os atributos visuais. Neles, estão contidos elementos como fontes, paleta de cores e formas.

https://cwi.com.br/blog/kits-de-ui-edite-e-planeje-a-aparencia-da-interface-do-seu-produto/


Um UI kit garante a consistência e coesão dos componentes de UI, atuando como referência para design de interfaces mobile ou web.

 

Em geral, esses kits costumam ser destinados a um nicho específico, como finanças, comércio ou música.

https://aelaschool.com/designdeinteracao/consistencia-e-padronizacao-em-ui/

Vantagens de um UI kit

Algumas das principais vantagens dos kits de interface do usuário são:

acelerar o processo de design

Com tudo pronto e agrupado, o time ganha mais celeridade para apenas utilizar os itens já prontos;

manter a consistência no projeto

Quando combinamos elementos derivados de diferentes bibliotecas, ou mesmo a partir da criação do próprio UI designer, pode ser difícil manter a consistência.

A adoção de UI kits otimiza a criação.



Desvantagens de um UI kit

Algumas das desvantagens dos kits de interface do usuário são:

quantidade limitada de elementos

Mesmo com tudo muito organizado, é importante dizer que os UI kits têm uma quantidade limitada de elementos;

pode necessitar de adaptação

Se existe a intenção de incorporá-lo a um design já existente, será necessário adaptar o kit para manter a consistência.
